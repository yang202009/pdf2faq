# PDF2FAQ

将 PDF 文档或文本转换为问答式学习 FAQ，帮助深度理解书籍、报告和文档。

## 功能特点

- **智能分析**：自动识别文档结构、章节和关键内容
- **来源追溯**：明确标注原始文档、实际分析对象和引用来源
- **教练式问答**：覆盖优先的深度学习 FAQ，而非简单摘要
- **证据标注**：区分原文明确说明、暗示推断和证据不足的内容
- **多种模式**：支持标准学习版和商业审计增强版

## 使用方法

### 基本用法

```
原始文档路径 + 阅读目标
```

示例：
```
D:\documents\book.pdf，帮我真正理解这本书的核心框架
```

### 完整用法

```
原始文档路径：path/to/original.pdf
实际分析对象路径：path/to/extracted.txt（可选）
阅读目标：帮我理解核心概念并提炼可用于写作的观点
输出模式：标准学习版（默认）或商业审计增强版
```

## 支持的文档类型

- PDF 文档（直接分析）
- 从 PDF 提取的 TXT/MD 文件
- 混合模式（PDF 原文 + TXT 分析对象）

## 工作流程

1. 读取文档并判断来源关系
2. 评估提取质量和结构完整性
3. 生成分析提纲并等待确认
4. 生成完整的 Markdown FAQ

## 输出模式

### 标准学习版
适用于理解、写作支持、学习笔记场景，包含：
- 文档来源说明
- 核心问题解答
- 证据标注
- 延伸思考

### 商业审计增强版
适用于需要更强可追溯性的场景，额外包含：
- 完整性声明
- 风险披露
- 审计摘要
- 克制版延伸思考

## 示例输出

生成的 FAQ 包含：
- 文档处理说明（原始文档、分析对象、来源关系）
- 结构化问答（按主题分组）
- 每个回答包含：核心洞察、证据支持、局限性、延伸思考
