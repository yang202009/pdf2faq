# Source Integrity Rules

Use these rules whenever the input chain is anything other than a clean, directly readable original file.

## Hard rules

- Always distinguish **原始文档** from **实际分析对象**.
- If the analysis is based on TXT / MD derived from a PDF, say so explicitly.
- Never present TXT-derived evidence as if it were directly verified from the original PDF unless the PDF was actually checked.
- Never invent page numbers or imply page precision that the extraction does not support.
- If a claim is synthesized from multiple passages, label it as `综合推断`.
- If the document omits visuals, tables, footnotes, or layout cues, disclose that limitation when it affects confidence.
- If anchor quality is weak, lower confidence instead of writing around the weakness.

## Quick self-check

Before final output, confirm:
- Did I state the original document?
- Did I state the actual analysis object?
- Did I describe the source relationship honestly?
- Did I avoid fake provenance?
- Did I avoid fake citation precision?
- Did I label inference vs explicit support?
